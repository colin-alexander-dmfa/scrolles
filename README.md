# Scrolles (WIP)
A deadly simple scroll progress indicator based on css and js.

> Please note this project is in work in progress.

## Installation
```sh
yarn add -D scrolles
```

## Usage (temp)

```js
scrollProgress({
   mode: 'continuous', // or 'steps'
   reverse: false, // or true
   style: {
	color: 'paleblue',
	height: '3px'
   }
});
```
